#  [Template] KMP Web With Compose

Template of a Kotlin Multiplatform project with Kotlin/JS target and Compose Multiplatform.

## Versions
- Compose: 1.5.3
- Kotlin: 1.9.0
- Gradle: 8.2
- JDK: 19
